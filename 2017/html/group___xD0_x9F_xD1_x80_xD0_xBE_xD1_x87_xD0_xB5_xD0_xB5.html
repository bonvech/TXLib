Рисует часовню

Аргументы
xx-абсцисса центра круга, лежащего внутри полигона ( часовни ).
yy-разность ординаты нижнего левого угла полигона ( часовни ) и 100.
scalescale-коэффицент размера часовни и ее составляющих.
colorcolor-возможный цвет полигона ( часовни ) ( полностью ) ( смотр. t ).
colortcolort-цвет круга, внутри которого лежит круг, лежащий внутри полигона ( часовни ) ( без контура ) ( на вид ).
colortecolorte-цвет контура круга, внутри которого лежит круг, лежащий внутри полигона( на вид ).
colorncolorn-возможный цвет полигона ( часовни ) ( полностью ) ( смотр. t ).
ll-радиус круга, внутри которого лежит круг, лежащий внутри полигона ( часовни ) ( на вид ), при scale=1.
qq-разность абсциссы точки, являющийся концом нижней ( при {q, p, z}=0 ) линии ( из двух, смотря по необщему концу), другим концом которой явлется центр круга, лежащего внутри полигона ( часовни ) и [суммы x и 5] , при scale=1.
pp-разность абсциссы точки, являющийся концом верхней ( при {q, p, z}=0 ) линии ( из двух, смотря по необщему концу), другим концом которой явлется центр круга, лежащего внутри полигона ( часовни ) и x , при scale=1.
gg-толщина контура круга, внутри которого лежит круг, лежащий внутри полигона ( часовни ) ( на вид ).
ww-разность y и [суммы ординаты высшей точки полигона ( часовни ) и {суммы 50 и z}] , при scale=1.
zz-разность y и ординаты верхней правой точки полигона ( часовни ), при scale=1.
tt-параметр, при значении которого меньше 1000, цвет полигона ( часовни ) становиться colorn, а при другом значение - color.
Заметки
x0=325,y0=500
Предупреждения
Важно:
-6<q<6;
-6<p<6.
drawChasovnia(425,400,0.5, TX_BROWN, TX_YELLOW, TX_BLACK, TX_GREEN, 30, 5, -3, 2, 10, 10, 2);

Аргументы
xx-абсцисса верхней точки полигона.
yy-ордината верхней точки полигона.
scalescale-коэффицент размера колпака и его составляющих.
colorcolor-цвет полигона ( полностью ).
coloricolori-цвет круга, внутри которого лежит круг и центром, которго является точка с координатой x,y, при s<50.
coloracolora-цвет лучей ( линий находящихся рядом с полигоном ).
colorucoloru-цвет нижнего круга ( без контура ).
ee-радиус круга, внутри которого лежит круг и центром, которго является точка с координатой x,y, при s<50, при scale=1.
dd-толщина контура нижнего круга, при scale=1.
ww-разность x и [суммы абсциссы точки, являющийся ближним к полигону концом левого луча и 75], при scale=1.
jj-разность радиуса круга, внутри которого лежит круг и центром, которго является точка с координатой x,y, при s<50 и 50, при scale=1.
ss-разность радиуса круга, который лежит внутри круга и центром, которго является точка с координатой x,y, при s<50 и j, при scale=1.
ff-параметр при четном значении которого, его абсциссой центра нижнего круга является сумма x и f, а при нечетном - разность.
drawKolpak(250,400,0.5, TX_GREEN, TX_LIGHTCYAN, TX_BLACK, TX_WHITE, 20, 5, 10, -30, 60, 8);
