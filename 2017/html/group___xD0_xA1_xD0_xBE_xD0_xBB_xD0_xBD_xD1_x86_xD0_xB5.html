Рисует Солнце

Аргументы
xx-абсцисса центра круга ( солнца ).
yy-ордината центра круга ( солнца ).
scalescale-коэффицент размера солнца и его состовляющих.
colorcolor-цвет круга ( солнца ) и его лучей ( полностью ).
colorbcolorb-контура круга, лежащего внутри круга ( солнца ).
colorncolorn-цвет круга, лежащего внутри круга ( солнца ) ( без контура ).
colorvcolorv-цвет контура круга, лежащего внутри круга, лежащего внутри круга ( солнца ).
colorccolorc-цвет круга, лежащего внутри круга, лежащего внутри круга ( солнца ) ( без контура ).
coloracolorp-цвет круга, внутри которого лежит круг ( солнце ) ( без контура ).
coloracolorp-цвет контура круга внутри, которого лежит круг ( солнце ).
ee-разность абсциссы нижнего правого луча солнца и 50.
tt-частное размера круга, внутри которого лежит круг ( солнце ) и scale.
cc-частное размера круга, лежащего внутри круга ( солнца ) и scale.
ff-толщина контура круга, лежащего внутри круга, лежащего внутри круга ( солнца ).
ss-толщина контура круга ( солнца ).
rara-параметр, который меняет цвет круга ( солнца ) и лучей на красный, а цвет круга внутри которого лежит круг ( солнце ) - на черный, при его значении равном 15.
Предупреждения
(e, s, f при scale=1)
drawColnce(500,500,1, TX_YELLOW, TX_BROWN, TX_MAGENTA, TX_LIGHTCYAN, TX_BLACK, TX_BLUE, TX_BLUE, 0, 0, 0, 1, 1, 0);
