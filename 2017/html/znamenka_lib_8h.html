Птичка Чирикиевна Рисует птицу.

Аргументы
xX-координата опроной точки
yY-координата опорной точки
sпараметр размера
colorцвет тела и клюва
color1цвет крыла
hПодъём/опускание крыла(в недвижимой позици), изменение координаты по Y-оси кончика крыла
kПодъём/опускание клюва(в недвижимой позици), изменение координаты по Y-оси кончика клюва
sleep_kпараметр глаза: при значении 1 - глаз открыт, 0 - закрыт
pпараметр поворота птицы (при отрицательном значении переменной - поворот вправо клювом, при положительном - влево)
drawBird(200, 150, 1, (RGB(0, 255, 204)), (RGB(0,51,51)), 10, 10, 1, -1);

Дом Рисует дом.

Аргументы
xX-координата опроной точки
yY-координата опорной точки
sпараметр размера
colorцвет крыши
color1цвет стен дома
color2цвет рамы окна, каёмки двери и самой двери
light_onпараметр цвета света в доме: при значении 1 - color3, 0 - color4
hпарметр высоты крыши
tпараметр ширины дома (влево от двери) (не рассширяет дверь)
lпараметр раскрытия двери
color3цвет света в доме при значении light_on 1
color4цвет света в доме при значении light_on 0
kпараметр наклона крыши: т.е. перемещение верней точки крыши по X-оси (при отрицательном значении перменной - вправо, при положительном - влево)
Предупреждения
Следует использовать значения переменной l (параметр раскрытия двери) в условиях 0 <= l <= 100.
drawHouse(100, 600, 0.5, (RGB (178, 34, 34)), (RGB (140, 80, 50)), (TX_BLACK), 1, 10, 20, 40, (TX_YELLOW), (TX_BLUE), -50);
