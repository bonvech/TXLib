Рисует дом

Аргументы
xположение по оси X верхней точки крыши
yположение по оси Y верхней точки крыши
scaleкоэффицент размера (при значении равном 1 ширина равна 300 высота 400)
color1отвечает за цвет крыши
color2отвечает за цвет стен
color3отвечает за цвет двери
widthприбавка к ширине(перемещает правый край направо на заданное число пикселей)
heightприбавка к высоте крыши(перемещает верхнюю точку вверх на заданное число пикселей)
light_onпараметр цвета окна(если равен 0, окно чёрное. Иначе жёлтое)
door_openпараметр открытия и закрытия двери(Если равен 0, дверь закрыта.Иначе открыта.)
high2отвечает за высоту всего дома(нижний край остаётся на месте.Остальные точки перемещаются вверх на заданное число пикселей)
drawHouse(100,700,1,TX_RED,TX_BLACK,TX_RED,0,0,1,0,0)

Рисует человека

Аргументы
xположение по оси X основания ног
yположение по оси Y основания ног
scaleкоэффицент размера(при значении равном 1 высота равна 250 пикселей, ширина 100)
colorотвечает за цвет(всего объекта)
zсмещение центра рта(при значении равном 140 рот является одним отрезком)
angleперемещение левой руки по вертикали(при слишком большом значении ломает рисунок)
eyes_closedотвечает за закрытие глаз(при значении больше 0 глаза закрыты)
angle_legsперемещение ног по горизонтали(при слишком большом значении ломает рисунок)
eyes_moveперемещение глаз по оси X(при слишком большом значении ломает рисунок)
angle2перемещение правой руки по вертикали(при слишком большом значении ломает рисунок)
drawHuman(500,500,1,TX_BLACK,140,0,0,0,0,0)

Рисует солнце

Аргументы
xположение по оси X центра солнца
yположение по оси Y центра солнца
scaleкоэффицент размера(при значении равном 1 радиус и длина лучей равны 100 пикселей)
colorотвечает за цвет(всего объекта)
lenghtкоэффицент длины лучей(при значении равном 1,длина лучей 100 пикселей)
radiusприбавка к радиусу
linesотвечает за существование или отсутствие лучей (если равно 0 лучи исчезают)
spinвращает солнце по кругу (работает не совсем так как надо)
onотвечает за существование объекта (если равно 0 объект исчезает)
drawSun(100,100,1,TX_YELLOW,1,0,1,0,1)
