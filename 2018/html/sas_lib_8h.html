Рисует инопланетянина(осьминога)

Аргументы
x- координата центра туловища (овала) по горизонтали
y- координата центра туловища (овала) по вертикали
scale- размер персонажа
color- цвет туловища персонажа
legslength- длина ног(щупалец) (стандарт (scale = 1, legslength = 1) - длина ног = 50)
eye- БИНАРНЫЙ ПАРАМЕТР! положение зрачка (взгляда) персонажа. 0 - в центре, 1 - вправо, 2 - влево
height- высота туловища персонажа отдельно от глаз и ног.
legsmove- БИНАРНЫЙ ПАРАМЕТР! движение ног. 0 - на месте, 1 - влево, 2 - вправо
eyessize- размер глаз отдельно от всего остального (стандарт (scale = 1, eyessize = 1) - размер глаза = 15, размер зрачка = 8)
draw_alien(300,700,1.0,TX_RED,1.0,1,1,0,1);

Рисует пакмана(пэкмэна)

Аргументы
x- координата центра пакмана (круга) по горизонтали
y- координата центра пакмана (круга) по вертикали
scale- размер пакмана (стандарт (scale = 1) - радиус круга = 50 )
color- цвет тела (круга) пакмана
mouth- БИНАРНЫЙ ПАРАМЕТР! открытие рта. 1 - закрыт, 0 - открыт
eyex- координата глаза по горизонтали
eyey- координата глаза по вертикали
eyesize- размер глаза (стандарт (eyesize = 1 и scale = 1) - радиус круга = 5 )
reversecharacter- БИНАРНЫЙ ПАРАМЕТР! поворот персонажа влево или вправо. 1 - влево, 0 - вправо
Предупреждения
50<eyex<-10, 50<eyey<0
draw_pacman(100,700,1.0,TX_YELLOW,1.0,1.0,0,0,1,1);
